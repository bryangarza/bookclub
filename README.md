# bookclub

Website for Hacker Book Club

## Installation

Download from https://github.com/frankcash/bookclub.

Then run:

```
$ lein deps
```

## Running

```
$lein repl

=> (use 'ring.adapter.jetty)
=> (use 'hello-world.core)
=> (run-jetty handler {:port 300})
```
